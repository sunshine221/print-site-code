# 需求卡片：FE-003 产品与案例库（列表+详情）

## 基本信息
- 编号：FE-003
- 名称：产品与案例库（列表+详情）
- 状态：draft
- 关联文档：
  - PRD：`.trae/documents/3D打印产品展示与管理平台-PRD.md`
  - 技术架构：`.trae/documents/3D打印产品展示与管理平台-技术架构.md`

## 背景与目标
- 背景：需要一个“可筛选可检索”的产品/案例展示库，承载 SKU 信息、参数、参考价与媒体内容，支撑询价转化。
- 目标用户/使用场景：访客浏览列表→进入详情→选择 SKU→发起询价（产品路径）。

## 范围
- In scope：
  - 列表页 `/work`：卡片网格、主图、关键参数摘要、筛选/搜索（可逐步实现）。
  - 详情页 `/work/:id`：媒体画廊、参数与说明、SKU 展示、行动按钮“立即询价”。
  - 公开 API 支撑：`GET /api/products`、`GET /api/products/:id`（案例可后续补齐为 `/api/cases`）。
- Out of scope：
  - 3D 预览（可选增强）。
  - 收藏/下载资料（可选增强）。

## 用户故事
- 作为访客，我想通过筛选与搜索快速找到合适产品，并查看其规格与参考价，从而决定发起询价。

## 验收标准（必须可测试）
- [ ] 访问 `/work` 能看到产品列表，包含主图与标题等核心信息。
- [ ] 支持至少一种筛选或搜索（例如关键字搜索）；无数据时有清晰空态。
- [ ] 点击列表项可进入 `/work/:id`，详情页可渲染媒体与基础参数。
- [ ] 详情页“立即询价”可跳转到 `/quote?type=product&skuId=...` 或等价的产品询价入口（具体参数在 Tech Spec 定稿）。
- [ ] 列表与详情的 loading/error/empty 状态完整，失败不导致白屏。

## 边界与反例用例
- 空状态：无产品时显示空态，并引导返回首页或联系咨询。
- 异常：详情 id 不存在时给出 404 提示并提供返回入口。
- 图片：图片缺失时使用占位展示，不破坏布局。
- 性能：列表分页/限制数量；避免一次性渲染过多。

## 依赖与约束
- 相关入口：
  - 页面：`workspace/src/pages/WorkList.tsx`、`workspace/src/pages/WorkDetail.tsx`
  - 路由：`workspace/src/App.tsx`
  - API：`workspace/api/routes/products.ts`

## 实现拆分（最小可回滚）
- 子任务 1：列表展示（基础字段 + loading/error/empty）。
- 子任务 2：详情展示（媒体 + 参数 + CTA）。
- 子任务 3：筛选/搜索与分页（逐步增强，先简单后复杂）。
- 子任务 4：SKU 选择与询价入口参数规范（与 FE-004 对齐）。

## 验证方式
- 自动化：`npm run check`、`npm run lint`、`npm run build`
- 手动路径：
  - 访问 `/work`，检查列表、空态、错误态
  - 点击进入 `/work/:id`，检查详情渲染与 CTA

## 发布与回滚
- 发布检查点：列表/详情可访问，关键 CTA 可跳转，错误态不白屏。
- 回滚方式：如仅前端变更可直接回滚；如涉及 API/DB 变更需同时回滚后端与迁移（写入发布检查单）。
